## Informações

**Sobre este projeto:**

- Uma SPA simples com login, autenticação, requisições e animações.

**Funcionalidades:**

- Fazer requisições GET, POST, PUT, DELETE,
- Fazer requisições com autenticação de token.
- Token expirado retorna para tela de login.
- Tema Claro e Tema Escuro (salvos no localStorage) 

**Tecnologias:**

- **Front-end:** React + Redux.
- **API Rest:** Axios.
- **Outras Libs/APIs:** React Helme + React Router Dom.
- **Pré-processador:** SASS.



## Como rodar

**Requisitos:**

- Ter instalado o Node.Js e o NPM [Download](https://nodejs.org/en/download/).

**Download das dependências:**

- No diretório raiz use o comando `npm install` para fazer o download das dependências.

**Como rodar:**

- Para iniciar o servidor no modo "Desenvolvedor" use o comando : `npm start`.
- Para realizar o build dos arquivos use o comando: `npm run build`.
